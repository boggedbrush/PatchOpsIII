Features

Everything you need, in one cohesive UI.

PatchOpsIII consolidates popular community tools and quality-of-life tweaks for Black Ops III, with safe configuration handling and clear logging.

Download latest Quick start

Dashboard

A real-time status overview and launch control center in one glance.

  • Status overview for T7 Patch, DXVK, Enhanced, Reforged, and launch options.
  • Quality of Life toggles: skip intros, use latest d3dcompiler.
  • Launch profiles: Default, Play Offline, All-around Enhancement Lite, Ultimate Experience Mod, Reforged.
  • Smart Black Ops III directory detection (automatic + manual browse).

T7 Patch

Secure your multiplayer sessions with one-click patch management.

  • Install or update T7 Patch to mitigate known remote code execution vulnerabilities.
  • Set your gamertag and choose from multiple gamertag colors.
  • Configure a network password with Friends Only mode support.
  • Clean uninstall when needed.

BO3 Enhanced

Install and manage the Enhanced mod that unlocks higher-fidelity assets.

  • Install or update BO3 Enhanced from a local game dump.
  • Step-by-step guidance for providing a UWP game dump.
  • Clean uninstall preserves your base game files.

BO3 Reforged

Deploy the Reforged overhaul mod with full T7 Patch compatibility.

  • Install Reforged (also installs the BO3 Reforged Workshop mod).
  • Force Ranked Mode and Steam Achievements toggles.
  • Network password synced to players/T7.json for cross-compatible multiplayer.
  • Works with or without T7 Patch; cross-compatible when both players use Reforged.

Graphics

Fast, practical graphics controls — from curated presets to your exact refresh rate.

  • Preset loader for curated JSON configurations (e.g. Smooth Preset).
  • FOV slider, display mode, resolution, refresh rate, render resolution %, and V-Sync.
  • FPS limiter (0–1000) and FPS counter toggle.

DXVK-GPLAsync

Reduce shader stutter with async Vulkan compilation tuned for BO3.

  • Install or uninstall DXVK-GPLAsync with one click.
  • Configure async shader compilation, GPL async cache, and FPS/GPU HUD.
  • Tune compiler threads, frame rate cap, frame latency, and tear-free mode.
  • Presets for quick recommended configuration.

Advanced

Unlock hidden settings and apply power tweaks while keeping your config safe.

  • SmoothFramerate, VRAM target limit, frame latency, and Reduce CPU Usage toggles.
  • Unlock all graphics options and set config.ini read-only.
  • Clear logs, copy logs, clear mod files, and reset to stock.
  • Guarded writes that reduce accidental overwrites of user settings.

Terminal & logging

Every operation prints live feedback, and a persistent PatchOpsIII.log helps with troubleshooting and support.

Live feedback

See what succeeded (or failed) without guessing — especially during installs and patching.

Auditable logs

A detailed log file captures actions for reproducible bug reports and faster support.

Cross-platform

Built and packaged for both Windows and Linux, including Steam Deck-friendly releases.
